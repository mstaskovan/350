Setting up your Flight Deck - pattern flow -- overhead ↑↑↑ -- ECAM ↓↑ -- pedestal ↓↓↓ -- FCU glareshield →

  • Crew supply on
  • RCDR GND CTL
  • EXT LT strobe auto, beacon off, NAV pos 1, LOGO auto, WING off, LDG off, NOSE off
  • SIGNS seat belts on, no smoking auto, no mobile auto, emer exit light arm
  • FUEL pump left to right on
  • ISIS baro set
  • A brk off, anti skid on
  • RMP check again
  • Thrust lever idle, eng master lever off, eng mode selector norm
  • TCAS
  • RMP 3 to squawk
  • Ckpt door to norm

Programming the FMS

  • ATC COM bt - ADS/C connected
  • MSG RECORD pg - deleted
  • SURV bt - XPDR stby
  • INIT bt - check according FTL plan CRZ FL, temp, tropo, WIND request
  • IRS
  • Departure - insert departure and then crosscheck on ND and charts
  • Insert arrival then crosscheck on ND and charts
  • Navaids
  • FUEL and LOAD set ZFW, ZFWCG, BLOCK, PAX NBR, check and set FUEL according FLT plan
  • T.O. perf sent from OIS and confirm, packs set manually and T.O. shift also manually, check THR RED, Accel, trans alt, eo accel set
  • Move to CLB and check PRESEL speed from there to POS MONITOR set DIST TO departure RWY
  • F-PLN bt then F-PLN INFO set FIX INFO for EOsid procedure
  • SEC F-PLN bt
  • SEC1 EOsid procedure - click dep RWY and choose INSERT NEXT WPT, add manually DEP RWY/HDG/NM and set arrival
  • SEC2 - only copy active FPL
  • Back to F-PLN if is set STEP CLIMB, click to WPT and choose STEP ALT, set WPT and alt
  • After that, set FMS to PF site - ACTIVE/PERF and PM site - ACTIVE/F-PLN
  • Set FCU (glareshield) baro, CSTR, alt, ...
  • OXYGEN MASK test

Takeoff Briefing and BEFORE START CHECKLIST

  • check max ldg weight 207t check for EOsid precedure
  • c/l menu bt - before start down to the line

Starting the engines

  • When boarding complete, check door and loadsheet update fuel and load page, check GWCG
  • T.O. perf again on OIS, send to FMS and confirm
  • Disconnect ext power on overhead and also in OIS ground page, remove chock and arm sliding, check ECAM
  • Beacon turn ON
  • Surv pg - default settings
  • If Amber flight !!!ACARS!!!
  • C/L menu bt - before start below to the line activate GSX push-back, then if on Vatsim call for clearance push and start
  • Set display OIS and ND for taxi and call pushback service
  • Eng mode selector to IGN/START and eng 1 start lever ON, when avail then eng 2, eng must run 30 sec on idle
  • Eng after start flow - eng mode selector to NORM, APU bleed off, wing and eng anti-ice (if needed), APU off
  • Check ECAM status - arm splrs, rudder trim reset bt, set flaps, check trim on ND L=R
  • C/L menu bt - after start

Taxi to HP rwy

  • FLT control check, if on Vatsim call for clearance for taxi
  • Clear left side and right side
  • Nose light to taxi
  • Release PRK brake
  • If using thrust, no > 30% N1
  • BRK check also on ECAM
  • Taxi camera on
  • Strobe ON if crossing RWY
  • Taxi flow - auto brake ON, check squawk, TO config BT
  • C/L menu BT - before takeoff down to the line

from HP rwy, lineup and Takeoff

  • When ready to lineup perform lineup flow ANF off, range area out off taxi mode, OIS set for SID chart
  • Packs set
  • Strobe ON
  • IF received takeoff clearance, set Nose light to TO and LDG ON
  • WX on, TERR on if needed
  • TCAS set ABV
  • C/L menu BT - before takeoff below to the line
  • Clear left side and right side, lineup RWY and full stop on center line
  • Takeoff - engine 25%, half forward side stick, 3 deg per 1 sec, then 12.5%; positive climb gear up, after 5 sec AP 1 ON
  • When S speed - flaps 0, disarm SPLR, Nose LGT off, Pack off if needed
  • C/L menu BT - after takeoff down to the line

Climb

  • After takeoff flow - to near 10,000 ft, flaps up, speed brake retracted, gear up, packs on
  • Climb flow when reaching 10,000 ft - landing light off, seat belts auto
  • EFIS options - range, TERR, ...
  • ECAM memo check
  • FMS check - Navaids | clear manually added | PERF check optimum ALT and max, and compare with FLP
  • Check transition altitude and set BAROS to STD
  • C/L menu BT - after takeoff below to the line
